# Pipeline Integrity Management System (PIMS)

## Overview

A professional engineering dashboard for pipeline corrosion assessment, remaining life prediction, risk evaluation, and inspection management.

## Features

- Corrosion Rate Calculation
- Remaining Life Prediction
- API 570 Inspection Scheduling
- ASME B31.3 Minimum Thickness
- Risk Matrix
- Interactive Dashboard
- PDF Inspection Reports
- CSV/Excel Import & Export
- Pipeline Health Score

## Technology

- Python
- Streamlit
- Plotly
- Pandas
- NumPy
- ReportLab
